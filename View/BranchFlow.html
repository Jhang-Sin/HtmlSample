GitHub Pages 部署分支說明

這個架構說明 React 專案在 GitHub 上的兩個分支 main 與 gh-pages 的用途與流程。

main 分支

主要開發分支
包含 React / TypeScript 原始碼
及 README.md

手動推送：
git push origin main

➡️

npm run build

將程式打包至 dist/ 資料夾

➡️

gh-pages 分支

自動部署分支
用於 GitHub Pages 網站展示

自動推送：
npm run deploy

✅ 開發重點：
- 平常都在 main 分支開發。
- 每次更新完程式後，執行： 
npm run build && npm run deploy
- 系統會自動將靜態檔案推送到 gh-pages 分支，供 GitHub Pages 使用。
