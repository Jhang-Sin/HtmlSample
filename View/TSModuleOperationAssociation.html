TypeScript Import / Export 運作關係教學

📘 一、單一模組範例

檔案 1： src/models/User.ts

export class User {
  constructor(
    public name: string,
    public age: number,
    public gender: string
  ) {}
}

檔案 2： src/main.ts

import { User } from './models/User';

const u = new User('Alice', 25, 'F');
console.log(u);

📖 三、運作說明

  • export： 將模組內的類別、函式或變數公開給外部使用。
  • import： 從其他模組引入先前 export 的內容。
  • 模組（Module）： 每個 .ts 檔案都是一個模組，互相之間可透過 import/export 溝通。
  • 對應規則： import { User } from './models/User' 會自動尋找同路徑下的 User.ts 或 User.js。

🧩 四、多模組進階範例

檔案 1： src/models/User.ts

export class User {
  constructor(public name: string, public age: number) {}
}

檔案 2： src/models/Product.ts

export class Product {
  constructor(public id: number, public title: string, public price: number) {}
}

檔案 3： src/main.ts

import { User } from './models/User';
import { Product } from './models/Product';

const user = new User('Bob', 30);
const product = new Product(1, 'Keyboard', 1200);

console.log(user);
console.log(product);
